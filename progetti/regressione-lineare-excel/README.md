# Dati casuali e regressione lineare (Statistica)

Progetto Excel di statistica applicata: generazione di dataset casuali, analisi statistica esplorativa e applicazione di un modello di regressione lineare.

## Obiettivo

Esplorare tecniche di generazione dati, campionamento, analisi della correlazione tra variabili e modellazione tramite regressione lineare, direttamente in Excel.

## Struttura del file

- **Indice** — pagina di navigazione tra i fogli
- **Parameters** — parametri di configurazione per la generazione dei dati
- **Data** — dataset generato casualmente
- **Sample_filter** — esempio di campionamento con filtro
- **Sample_if** — esempio di campionamento condizionale
- **Statistical Insight** — statistiche descrittive sul dataset
- **(Un)correlated Variables** — analisi della correlazione tra variabili
- **Linear Regression** — applicazione del modello di regressione lineare

## Tecnologie

Excel — funzioni statistiche, generazione dati casuali, analisi di correlazione, regressione lineare
