# Report punti vendita — TechMarket (Power BI)

Report interattivo in Power BI sui punti vendita di una catena di distribuzione di elettronica, con analisi su vendite, prodotti, negozi e resi.

## Obiettivo

Fornire una visione d'insieme sulle performance di vendita della catena, per supportare decisioni su assortimento prodotti, gestione punti vendita e monitoraggio dei resi.

## Struttura del report

- **Indice** — pagina di navigazione tra le sezioni del report
- **Vendite per Mese** — andamento delle vendite nel tempo
- **Unità Vendute per Città** — distribuzione geografica delle vendite
- **Dettaglio Prodotto** — analisi a livello di singolo prodotto
- **Informazioni sui Negozi** — statistiche per punto vendita
- **Resi** — analisi dei resi prodotto

## Tecnologie

Power BI Desktop — modello dati, misure DAX, visualizzazioni interattive con filtri incrociati tra pagine
